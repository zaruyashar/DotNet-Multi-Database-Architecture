# Transp - Logistics & Cargo Management System

A comprehensive web application developed for a logistics company. It features real-time cargo tracking, dynamic shipping cost calculation, and an administrative dashboard for full operational control.

## 🚀 Technologies Used

* **Database:** MongoDB - NoSQL
* **Backend:** ASP.NET Core, C#
* **Frontend:** HTML/CSS/JavaScript
